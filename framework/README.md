## ¿Cómo ejecutar?

1. Instalar las dependencias

```bash
pip install -r requirements.txt
pip install uvicorn
```

2. Iniciar el servidor de FastAPI

```bash
python -m uvicorn main:app --reload

```

3. Iniciar la aplicación React

```bash
cd app
npm i
npm start
```
